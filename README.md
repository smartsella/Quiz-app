# Quiz App (HTML + CSS + JavaScript)

A simple and interactive Quiz Application built using **vanilla HTML, CSS, and JavaScript**.  
Users can select answers, view correct/incorrect responses, and see their final score.  
The app is clean, responsive, and easy to extend with more questions.
